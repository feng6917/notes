- 折叠所有方法

  `ctrl + k + 0`

- 打开(所有)折叠终端

  `ctrl + (k) + j`

- 复制一行（向上或向下）

  `alt + shfit + up/down`

- 单行注释

  `ctrl + /`

- 多行注释

  `alt + shift + a`

- 移动一行

  `alt + up/down`

- 显示/隐藏左侧目录栏

  `ctrl + b`

- 删除当前行

  `ctrl + shift + k`

- 控制台终端显示或隐藏

  `ctrl + ~; ctrl + j`

- 查找文件/安装 vscode 插件地址

  `ctrl + p`

- 代码格式化

  `shift + alt + f`

- 新建一个窗口

  `ctrl + alt + n`

- 行增加/减少缩进

  `ctrl + [/]`

- 字体放大/缩小

  `ctrl +/-`

- 拆分编辑器

  `ctrl + \`

- 关闭(所有)窗口

  `ctrl + (k) + w`

- 切换全屏

  `F11`

- 快速回到顶部/底部

  `ctrl + Home/End`

- 多行同时添加内容(光标)

  `ctrl + alt + up/down`

- (全局)替换

  `ctrl + (shift) + h`
